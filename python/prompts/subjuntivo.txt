Generate Spanish present subjunctive speech drill exercises.

TENSE: subjuntivo (presente de subjuntivo)
CONJUGATION RULES:
- Regular -ar → -e endings: hable, hables, hable, hablemos, habléis, hablen
- Regular -er/-ir → -a endings: coma, comas, coma, comamos, comáis, coman; viva, vivas, viva, vivamos, viváis, vivan
- Irregular: sea, esté, tenga, vaya, haga, diga, venga, pueda, quiera, sepa, ponga, salga, oiga, dé, haya
- Stem-changing: same as present indicative in singular + ellos (quiera, pueda, duerma, pida)

TRIGGER PHRASES (use diverse mix, these introduce the subjunctive):
espero que, quiero que, es necesario que, es importante que, ojalá, dudo que, no creo que, es posible que, antes de que, para que, cuando (future meaning), aunque, hasta que, a menos que, sin que

EXERCISE FORMAT:
Each exercise must have these 6 fields:
- id: "{verb}-subjuntivo-{number}" (number will be assigned later, use 0)
- verbo: the infinitive
- contexto: trigger phrase + subject + complement (comma-separated fragments showing the elements)
- respuesta: complete sentence with the trigger phrase and correct subjunctive form
- pista: a hint about why subjunctive is needed (wish, doubt, necessity, purpose, etc.)
- grupos: array of arrays, each inner array has the key idea unit

CRITICAL RULES FOR grupos:
1. Every element in grupos must appear verbatim in respuesta (after accent removal and lowercasing)
2. grupos elements must NOT contain accented characters — use unaccented forms only (e.g., "hable" not "hablé")
3. The conjugated subjunctive verb form is always its own grupo
4. The trigger phrase (espero que, es necesario que, etc.) should be its own grupo
5. Include alternative speech recognition forms where relevant

Generate 3 exercises per verb. Use different trigger phrases and subjects across the 3 exercises.

OUTPUT: Valid JSON array of exercise objects. Nothing else.
